#Art of Akshay — Portfolio Website

A portfolio site for Akshay, a BFA student working across digital illustration, ink, and watercolor.

Live site: art-of-akshay.netlify.app

Features:
Category/series-based gallery view
Lightbox for viewing individual pieces
Dark mode toggle — respects system preference on first visit, remembers your choice
Fully responsive layout
No build step — plain HTML, CSS, and JS


Tech Stack:
HTML5
CSS3 (custom properties for theming, no framework)
Vanilla JavaScript (no dependencies)
Deployed on Netlify


Running Locally:
This is a static site with no build process. Clone the repo and open index.html directly in a browser, or serve it locally
